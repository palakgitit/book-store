# 📚 Book Store Management System

A full-stack web application built using **Node.js, Express.js, MongoDB, EJS, and Multer** to manage book records with complete CRUD operations.

---

## 🚀 Live Demo

👉 https://your-live-link-here.com

---

## 📌 Features

- ➕ Add new books with image upload  
- 📖 View all books in list format  
- ✏️ Edit book details  
- ❌ Delete books  
- 🖼️ Upload and display book cover images  
- 📊 Dashboard with total books & quantity  
- 🎨 Clean UI with modern brown theme  

---

## 🛠️ Tech Stack

- Backend: Node.js, Express.js  
- Database: MongoDB + Mongoose  
- Frontend: EJS Templates, HTML, CSS  
- File Upload: Multer  
- Styling: Custom CSS
